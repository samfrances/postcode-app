# README

This app provides a GUI for the [postcodes.io](https://www.postcodes.io) API.

## React Features

- `useEffect` hook
- Custom hooks
- Routing with [React Router](https://reactrouter.com/)
- Typescript typing
- Type validation of data fetched from the API, with unit tests
